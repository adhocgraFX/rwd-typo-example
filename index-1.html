Typography

The typography is designed with a harmonical modular scale within its basic styles. All font sizes of paragraphs and headlines are set relative to the default html size of 100% = 16px. The line height is set on a 1.5 base for paragraphs and 1.2 for headlines.

Regular paragraphs on small media sizes are set at 100% = 16px, on medium screen sizes at 112,5% = 18px and on larger screen sizes at 125% = 20px. Thereby, they behave on different screen sizes responsive but maintain their visual appearance and line-heights.
